# 📚 Exercícios em Java

Este repositório foi criado com o objetivo de **registrar exercícios em Java** ao longo do meu processo de aprendizado.

A ideia é acompanhar minha evolução na linguagem, praticar conceitos de programação e construir uma base sólida em desenvolvimento.

## 🎯 Objetivo

- Praticar lógica de programação
- Aprender e aplicar conceitos da linguagem Java
- Registrar minha evolução ao longo do tempo
- Organizar exercícios resolvidos durante os estudos

## 🧠 Conteúdos praticados

Entre os temas estudados estão:

- Estruturas básicas da linguagem
- Programação orientada a objetos (POO)
- Classes e objetos
- Encapsulamento
- Construtores
- Métodos getters e setters
- Exercícios de lógica

## 🚀 Observação

Este repositório faz parte do meu processo de aprendizado, então os códigos podem evoluir e melhorar com o tempo conforme eu aprendo novos conceitos.
